# Pro-Recruiter-AI
Pro-Recruiter AI ayuda a los reclutadores a filtrar cientos de CVs en segundos. Utiliza el poder de los modelos de lenguaje de Groq (Llama-3) para comparar automáticamente las habilidades de un candidato contra la descripción de un puesto específico, generando una puntuación de compatibilidad, un análisis detallado y un grafico con las habilidades.
